# Need

<p><span class="p1">Need</span> <b>виражає</b>:</p>


<span class="p1">Необхiднiсть</span> (Necessity)			
<i>I <span class="p1">need</span> to go. - Менi <span class="p1">потрiбно</span> йти.</i>			
За допомогою модального дiєслова <span class="p1">need</span> ми показуємо необхiднiсть нашої дiї, менi необхiдно йти. <span class="p1">Need</span> має закiнчення <i>-s</i> в третiй особi однини.


В основному <span class="p1">need</span> використовується в питальних та заперечних реченнях:

<i>You don't <span class="p1">need</span> to clean the house. - <span class="p1">Не має</span> необхідності прибирати будинок.</i>

<i>Do I <span class="p1">need</span> to do this? - Чи є <span class="p1">необхідність</span> мені це робити?</i>

Для утворення заперечень та питань в реченнях з <span class="p1">need</span> вживаються допоміжні дієслова.

В 3-й особі однини <span class="p1">need</span> має закінчення <span class="p1">-s</span>.
